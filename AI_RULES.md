# AI_RULES.md

Rules for AI coding agents working on this repository.

## Principles
- Make small, focused changes.
- Preserve existing behavior.
- Prefer simple and readable solutions.
- Avoid broad refactors unless explicitly requested.
- Keep changes easy to review.

## Safety
- Do not add confidential values to the repository.
- Use existing project configuration patterns.
- Avoid changing generated or build output unless requested.

## Workflow
- Inspect relevant files before editing.
- Reuse existing patterns.
- Run available validation commands when possible.
- Summarize changes and mention anything not validated.

## Priority
Follow this file first, then `AGENTS.md`, then `CLAUDE.md`, then the specific task.
